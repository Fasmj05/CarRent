### Run Locally

To run **CarRent** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/Fasmj05/CarRent.git
```

Windows:

```bash
git clone https://github.com/Fasmj05/CarRent.git
```
